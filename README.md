# Louies-Revenge

**Louies-Revenge** is a fast-paced, **Galaga-style arcade shooter** built in **Python using Pygame**. The game focuses on classic arcade mechanics such as enemy waves, projectile combat, collision detection, and score tracking, all implemented from scratch within a real-time game loop.

---

## Gameplay Overview

- Player controls a ship
- Waves of enemies descend in classic arcade patterns
- The objective is to survive as long as possible while maximizing score

---

## Features

- Real-time arcade gameplay loop
- Enemy spawn system
- Player projectile system
- **Collision detection** between ships, bullets, and enemies
- Score tracking and lives system
- Keyboard-based movement and firing controls
- Modular architecture separating game logic, entities, and rendering

---

## Core Systems

- **Game Loop:** Handles input, updates game state, and renders frames at a fixed rate
- **Collision System:** Axis-aligned bounding box collision checks
- **Enemy AI:** Pattern-based movement
- **State Management:** Tracks score, lives, and game-over conditions
